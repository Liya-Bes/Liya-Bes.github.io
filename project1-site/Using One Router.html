Constructing a LAN Using One Router

LAN Setup: Step-by-Step

We built a LAN (Local Area Network) so two computers could share files and talk to each other.

Step 1: Reset the Router

We reset the router by pressing the small reset button then connect it to the laptop

Step 2: Pick a New IP Range

We logged into the router using a web browser and chose a new IP range for our group.

Our group used this range: 192.168.110.1 - 192.168.110.255

Step 3: Connect the Computers & Turn off WI-FI

We used Ethernet cables to connect both computers to the router. Each cable plugged into the back of the computer and into one of the router’s LAN ports.

Also, to make sure our test was fair, we turned off Wi-Fi on both computers.

TCP/IP Configuration

Data Transmission Experiment

We sent the file from Computer 1 to Computer 2 using the shared Samba folder.

Result

We opened “System Monitor” to see how fast the file was being sent.

Transfer Speed: About 67.7 Mbps

Time Taken: Around 1 minute and 25 seconds

Problems and Solutions

Problem 1:

One computer didn’t get an IP address at first.Couldn't access each others file

Solution:

We unplugged the Ethernet cable and plugged it back in. Then we restarted the laptop and checked that we turn off the wi-fi. we also manually searched for
eachothers IP address in Samba to be able to access the Folder in eachothers laptop. By doing this we were able to access and send each other big sized files. It worked!

Problem 2:

The file transfer was slow at first.

Solution:

We disabled Wi-Fi to make sure the computers used only the Ethernet cable.
