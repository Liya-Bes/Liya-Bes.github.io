Constructing a LAN Using Two Routers

LAN Setup: Step-by-Step

In this task, we made a LAN (Local Area Network) using two routers and two computers.

Step 1: Connect the Computers and Routers

We plugged Mark's Computer into Router 1 using an Ethernet cable.

We plugged Liya's Computer into Router 2 using another Ethernet cable.

Then, we connected Router 1 to Router 2 using a third Ethernet cable in the LAN ports.

Step 2: Configure the Routers

We logged into the router using a web browser and chose a new IP range for our group.

Our group used this range: 192.168.110.1 - 192.168.110.255

Step 3: Connect the Computers & Turn off WI-FI

We logged into each router with a web browser and set up two different IP address

Router 1 IP Range: 192.168.110.1 - 192.168.110.255

Router 2 IP Range: 192.168.110.2 - 192.168.111.255
